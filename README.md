tumblr-dl
=========

Yet another tumblr downloader based on Scrapy 

Installation :

```
$ pip install scrapy
$ git clone git://github.com/2xyo/tumblr-dl.git
```

Usage : 

```
$ cd tumblr-dl
$ scrapy crawl tumblr -a tumblr=http://securityreactions.tumblr.com path=/tmp/
```
